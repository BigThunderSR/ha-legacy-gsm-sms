All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.1.0] - 2025-06-12

### Added - Addon Support

- Home Assistant Addon support
- Addon configuration through the Supervisor UI
- Improved device detection in addon mode
- Automatic installation of dependencies in addon environment

## [1.0.0] - 2025-06-12

### Added

- Initial release
- Support for sending and receiving SMS messages
- Signal and network status sensors
- Home Assistant UI integration
- Compatibility with various GSM modems
